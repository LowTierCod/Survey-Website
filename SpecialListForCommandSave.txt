Репозиторий (Repo): Это просто твоя папка с проектом, за которой следит Git.

Commit (Коммит): Тот самый «снимок». Ты как бы говоришь: «Всё, я дописал кусок кода, зафиксируй это состояние».

Stage (Индекс/Зона ожидания): Представь это как корзину в супермаркете. Ты накидал туда файлы, которые хочешь сохранить в коммит, но еще не «оплатил» их на кассе.

Branch Команда git branch в Git — это основной инструмент для управления ветками: создания, просмотра, переименования и удаления веток в репозитории. Ветка в Git — это «скользящий» указатель на один из коммитов, позволяющий вести параллельную разработку, не затрагивая основной код.

Push (Пуш): Отправка твоих локальных коммитов в «облако» (на GitHub).
